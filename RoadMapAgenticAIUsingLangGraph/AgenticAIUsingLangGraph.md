## Curriculum Overview

### Important Disclaimer

The curriculum is flexible and subject to change due to the rapid evolution of agentic AI and LangGraph itself. The playlist will be modular rather than strictly topic-by-topic, allowing adaptation to new developments and learner feedback.

### Module 1: Foundations of Agentic AI

This module will include roughly five to six videos providing an in-depth overview of agentic AI. Topics include:  
- What is agentic AI?  
- Differences between AI agents and generative AI  
- What constitutes agentic retrieval-augmented generation (RAG)  
- Comparison between traditional RAG and agentic RAG  
- Key frameworks used for building agentic AI applications  

By the end of this module, learners will have a high-level understanding of the field and terminology.

### Module 2: LangGraph Fundamentals

The second module dives into the core concepts of LangGraph:  
- Building graphs  
- Understanding states, nodes, and edges  
- Conditional edges and looping mechanisms  

These fundamentals are essential for constructing AI workflows. The module also includes practical demonstrations on creating popular AI workflows using these concepts.

### Module 3: Advanced LangGraph Concepts

After grasping the basics, this module covers advanced features that enable industry-grade AI agents:  
- Persistence and memory management  
- Human-in-the-loop integration  
- Breakpoints and checkpoints  
- Time travel concepts in workflows  

Mastering these advanced topics will allow learners to build truly sophisticated and production-ready AI agents.

### Module 4: Building AI Agents

This is the most exciting module, where learners apply their knowledge to build different types of AI agents, including:  
- React agents  
- Reflection design pattern agents  
- Self-ask with help pattern  
- Planning-based agents  
- Multi-agent systems  

This hands-on experience will cement understanding and practical skills.

### Module 5: Agentic RAG Applications

Expanding on traditional retrieval-augmented generation, this module merges AI agents with RAG concepts to create agentic RAG architectures. Various architectures—including C-RAG and self-RAG—will be explored in detail.

### Module 6: Production and Deployment

The final module focuses on deploying AI agents as real-world projects fit for portfolios or interviews:  
- Building user interfaces for agents  
- Adding debugging and observability support  
- Integrating with LangChain  
- Deployment best practices  

Learners will complete a project that demonstrates their ability to develop and deploy agentic AI applications.

## Prerequisites for the Playlist

To maximise learning, the following prerequisites are recommended:

### Intermediate Python Skills

Learners should be comfortable with intermediate Python concepts including:  
- Object-oriented programming (OOP) principles  
- Typing modules and libraries like Pydantic  
- Asynchronous I/O programming  

Basic Python knowledge alone is not enough as advanced constructs will be used extensively.

### Familiarity with Large Language Models (LLMs)

A foundational understanding of LLMs is necessary, especially how to interact with and utilise them in AI workflows. If you have followed tutorials or playlists on LangChain or similar frameworks, you will find this playlist easier to follow.

### Experience with LangChain

Since LangGraph is built on top of LangChain, prior experience with LangChain is crucial. If you haven’t explored LangChain, it is strongly recommended to complete a LangChain course or playlist first to avoid struggling with dependencies and concepts.
